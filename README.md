# Object-Detection-on-NVIDIA-Jetson
This repository contains the workflow for the Object Detection inferencing on GPU edge device project for MENG 3540: Parallel Programming
# Pretrained Model Chosen: VGG16
This model was created by the Visual Geometry Group at Oxford in 2014, and the “16” in its name refers to its 16 layers with learnable and controllable weights. VGG16 is considered one of the best vision models of its time and is widely used as a feature extractor in many projects.

In total VGG16 has around 138 million parameters which is one of its biggest drawbacks requiring a lot of memory and computational power. Despite its size, VGG16’s ability to feature extract and accurately identify images makes it more than worth its memory commitment

# Requirements:
  - Windows 10/11 Computer
  - NVIDIA ORIN Jetson
  - Camera or Webcam
  - Python Compiler
  - Keras Libary
